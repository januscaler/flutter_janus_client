RtpStreamQuery

RtpStreamQuery class

Declarative matcher used to locate a specific RTP stats entry.

When multiple properties are supplied all of them must match. Provide historyKey or historyKeyResolver to avoid collisions when monitoring multiple tracks with the same mid. Set fallbackToFirstMatching to false if you prefer returning null instead of silently inspecting the first video stream when no explicit match is found.
