Calculates the bitrate of the inbound video stream identified by mid.

When mid is omitted, the first inbound video RTP stream is used. This retains the original string-based API while delegating the heavy lifting to mediaStats so developers can opt into richer statistics when needed.

Implementation

Future<String?> getBitrate([String? mid]) {
  return mediaStats.bitrateLabel(
    query: RtpStreamQuery(mid: mid),
    unit: BitrateUnit.kbps,
    fractionDigits: 0,
    suffix: 'kbps',
  );
}
