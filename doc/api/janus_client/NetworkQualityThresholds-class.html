NetworkQualityThresholds

NetworkQualityThresholds class

Classification thresholds that translate bitrate samples into NetworkQuality values.

The numeric values are expressed in kilobits per second. Lower each threshold to make the evaluator more tolerant on constrained networks, or increase them to demand higher quality before reporting "excellent". Ensure staleAfter is greater than your sampling interval so transient gaps do not instantly downgrade streams to stale.
